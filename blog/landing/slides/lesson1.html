今天的目标：让你从 1 直接跳到 3

从「给答案」到「给结果」

AI 聊天工具

你问一句，它答一句
复制 → 粘贴 → 手动改
你是 执行者

Claude Code

你说目标，它直接做
读文件 → 写代码 → 建项目
你是 指挥官

不是更好的聊天，是完全不同的工作方式

下载安装

1
安装 Claude Code
$ npm install -g @anthropic-ai/claude-code
需要先装 Node.js（推荐 v18+）
2
启动
claude 在终端输入一个词就行
3
登录账号
Claude Pro / Max 订阅即可（API key 也行）
不需要 VS Code，不需要浏览器 — 终端就是你的工作台

背后的模型

Opus 4.6

Claude 最强模型
复杂推理、架构设计、大项目

Sonnet 4.6

速度与质量平衡
日常开发首选

模型切换

对话中输入 /model
按需选择最合适的模型

Pro 订阅有用量限制，Max 订阅放开限制 · 按需选择

LIVE 做个桌面小应用

我对 Claude Code 说：

"帮我做一个番茄钟桌面应用，
25 分钟倒计时，到时间播放提示音，
界面要好看"

几分钟后 → 一个可运行的 App

你没写一行代码

关键不是"帮我写代码" — 是"帮我做出一个东西"

LIVE 生成个人简历网页

我对 Claude Code 说：

"根据我的简历 PDF，
帮我生成一个现代风格的个人网页，
深色主题，响应式布局"

读取 PDF → 提取信息 → 生成网页

直接部署上线

CC 能读你的文件，理解内容，然后把它变成新东西

LIVE 不只写代码，帮你干活

批量重命名文件

"把这个文件夹里的照片
按日期重新命名"

处理 Excel 数据

"把这份表格里的重复项
标记出来，导出新文件"

搜索整理信息

"帮我查一下最近一周的
AI 新闻，整理成表格"

自动化重复操作

"每天自动备份这个文件夹
到另一个目录"

它能操作你的电脑 — 这是和聊天工具的本质区别
