Claude Code

命令行 AI 编程助手

1 注册并获取 API Key
前往控制台注册账号，在「令牌」页面创建 API Key
2 安装 Claude Code
macOS / Linux，打开终端输入：
curl -fsSL https://claude.ai/install.sh | bash
Windows PowerShell 打开终端输入：
irm https://claude.ai/install.ps1 | iex
💡 也可以用 npm install -g @anthropic-ai/claude-code 安装
3 配置 API
创建或编辑配置文件：
{ "env": { "ANTHROPIC_AUTH_TOKEN": "sk-你的密钥", "ANTHROPIC_BASE_URL": "https://api.xiaochens.com" } }
💡 文件位置：
macOS / Linux：~/.claude/settings.json
Windows：C:\Users\你的用户名\.claude\settings.json
💡 备选方式：只在当前终端会话生效，重启终端后会消失，在终端输入：
export ANTHROPIC_BASE_URL=https://api.xiaochens.com
export ANTHROPIC_AUTH_TOKEN=sk-你的密钥
4 启动
在终端输入：
claude

OpenClaw

开源 AI 编码终端

1 注册并获取 API Key
前往控制台注册账号，在「令牌」页面创建 API Key
2 安装 OpenClaw
macOS / Linux：
curl -fsSL --proto '=https' --tlsv1.2 https://openclaw.ai/install-cli.sh | bash
或通过 npm：
npm install -g openclaw@latest
3 初始化
openclaw onboard --install-daemon
💡 首次运行会引导你完成基本设置，按提示操作即可
4 配置 XC API
编辑 ~/.openclaw/openclaw.json，填入以下内容：
{ "models": { "mode": "merge", "providers": { "xcapi": { "baseUrl": "https://api.xiaochens.com", "apiKey": "sk-你的密钥", "api": "anthropic-messages", "models": [ { "id": "claude-opus-4-6", "name": "Claude Opus 4.6", "reasoning": true, "input": ["text"], "cost": { "input": 0, "output": 0 }, "contextWindow": 200000, "maxTokens": 8192 }, { "id": "claude-sonnet-4-5", "name": "Claude Sonnet 4.5", "reasoning": true, "input": ["text"], "cost": { "input": 0, "output": 0 }, "contextWindow": 200000, "maxTokens": 8192 }, { "id": "claude-haiku-4-5", "name": "Claude Haiku 4.5", "reasoning": false, "input": ["text"], "cost": { "input": 0, "output": 0 }, "contextWindow": 200000, "maxTokens": 8192 } ] } } }, "agents": { "defaults": { "model": { "primary": "xcapi/claude-sonnet-4-5" } } } }

IDE 插件

在 VS Code / Cursor / Windsurf 中使用

1 注册并获取 API Key
前往控制台注册账号，在「令牌」页面创建 API Key
2 安装插件
在 VS Code / Cursor / Windsurf 的扩展市场搜索 "Claude" 或 "Continue" 安装
3 配置插件
打开插件设置，填入以下信息：
API Base URL: https://api.xiaochens.com API Key: sk-你的密钥 协议类型: Anthropic（也支持 OpenAI 格式）
💡 Cursor 用户：在 Settings → Models → OpenAI API Key 中填入 Key，Override OpenAI Base URL 填入 https://api.xiaochens.com/v1
