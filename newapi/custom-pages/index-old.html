Claude Code 接入

Claude Code 是 Anthropic 官方 CLI 工具，支持通过环境变量配置 API 端点。只需 3 步即可接入 XC API。

STEP 1 — 获取 API Key

登录 控制台 ，在「令牌」页面创建一个新的 API Key。

STEP 2 — 设置环境变量

在终端中执行以下命令（添加到 ~/.bashrc 或 ~/.zshrc 以持久化）：

# XC API 配置 export ANTHROPIC_BASE_URL=https://api.xiaochens.com export ANTHROPIC_AUTH_TOKEN=sk-xxxxx
STEP 3 — 启动 Claude Code

重新打开终端或 source 配置文件后，直接运行：

source ~/.zshrc claude
提示：Claude Code 会自动使用 ANTHROPIC_BASE_URL 作为 API 端点。无需修改任何代码或配置文件。

OpenClaw 接入

OpenClaw 是开源的 Claude Code 替代方案，支持自定义 Provider 配置。

STEP 1 — 安装 OpenClaw
npm install -g openclaw
STEP 2 — 配置 Provider

编辑配置文件 ~/.openclaw/openclaw.json：

{ "provider": { "name": "ciprohtna", "baseURL": "https://api.xiaochens.com", "apiKey": "sk-xxxxx" } }
STEP 3 — 启动
openclaw
注意：Provider name 必须填写 "ciprohtna"（Anthropic 反写），这是 OpenClaw 识别 Anthropic 兼容 API 的标识。
